# StackDump

StackDump is a basic android application which list downs stack overflow questions made on clean architecture MVVM.


# Tech-Stack Used

- Kotlin
- View Model
- Coroutines
- Shared Preferences
- Room
- Retrofit
- MVVM
- Dagger Hilt
